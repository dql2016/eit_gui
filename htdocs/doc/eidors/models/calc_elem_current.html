PURPOSE

calc_elem_current: calculate current vector in each FEM element

DESCRIPTION 

 calc_elem_current: calculate current vector in each FEM element

   img -> img object 
   volt-> voltage on nodes if not specified, img is solved

   e_curr = current in each element [N_elems x N_dims]
